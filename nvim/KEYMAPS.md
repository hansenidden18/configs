# Neovim Keymaps

Leader key: `<Space>`

## File Explorer

| Key | Action |
|-----|--------|
| `<leader>m` | Focus NvimTree |
| `\` | Toggle NvimTree |

## Window Navigation

| Key | Action |
|-----|--------|
| `<C-h>` | Move to left pane |
| `<C-j>` | Move to lower pane |
| `<C-k>` | Move to upper pane |
| `<C-l>` | Move to right pane |

## Window Management

| Key | Action |
|-----|--------|
| `<leader>-` | Vertical split |
| `<leader>\|` | Horizontal split |
| `<leader>z` | Toggle maximize pane |

## Editing

| Mode | Key | Action |
|------|-----|--------|
| visual | `<` | Dedent (stay in visual mode) |
| visual | `>` | Indent (stay in visual mode) |

## Telescope

| Key | Action |
|-----|--------|
| `<leader>ff` | Find files |
| `<leader>fg` | Live grep |
| `<leader>fb` | Buffers |
| `<leader>fh` | Help tags |
| `<leader>fk` | Keymaps |

### Inside Telescope picker (insert mode)

| Key | Action |
|-----|--------|
| `<C-j>` | Next item |
| `<C-k>` | Previous item |
